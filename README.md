
# Layout
iOS下的布局框架
=

前言
-

*  这是一个在IOS下面的布局框架，提供了流式布局，边框布局，卡片布局，表格布局，盒式布局，嵌套布局。；


一、概述
-
* 此框架设计于2012年完成，基于iPhone4进行设计，目前未进行维护
